Starting today, I will be doing the #100DaysOfCode challenge. It's a [coding challenge]{:target="_blank"} started by Alexander Kallaway. The reason I'm doing the challenge is to satisfy my desire to learn more. At my current web development job I feel I'm not learning the lastest JavaScript development methodologies and the latest frameworks. I also want to better my coding skills and apply that to my job. 

As per the main rules of the challenge, I will code minimum one hour per day for the next 100 days and tweet my progress (finally, I can use Twitter for something!). Feel free to follow me on [twitter]{:target="_blank"}.

If you are a web developer, or trying to become one, this is a great way to get motivated to code. Don't wait, start now!
